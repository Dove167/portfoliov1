# March 02, 2026 - Portfolio Deployment with Cloudflare, Fly.io & Mailgun

## Overview

Successfully deployed a full-stack portfolio application with a custom domain, SSL, and email functionality.

## Architecture

```
User → Cloudflare DNS → Fly.io (Astro + Rust/Axum) → Mailgun → Gmail
```

## Steps Completed

**Important:** Cloudflare proxy must be OFF (grey cloud) for Fly.io to handle TLS/SSL.

### 2. Fly.io Deployment

- App name: `josh-portfolio`
- Stack: Rust (Axum) backend + Astro frontend
- Database: SQLite (file-based)

#### Commands used:

```bash
# Login
fly auth login

# Check IPs
fly ips list -a josh-portfolio

# Add SSL certificate
fly certs add joshscompass.com -a josh-portfolio
fly certs add www.joshscompass.com -a josh-portfolio

# Verify certificate
fly certs check joshscompass.com -a josh-portfolio

# Deploy
fly deploy -a josh-portfolio
```

### 3. Mailgun Email Integration

#### What is Mailgun?
Mailgun is an email API service that allows developers to send, receive, and track emails programmatically.

#### Setup:
1. Signed up at mailgun.com
2. Created sending domain: `mg.joshscompass.com`
3. Got API key: `key-...`

#### Environment Variables:

```bash
# Local (.env)
MAILGUN_API_KEY=key-...
MAILGUN_DOMAIN=mg.joshscompass.com
CONTACT_EMAIL=jpfajar90@gmail.com
```

```bash
# Fly.io secrets
fly secrets set MAILGUN_API_KEY="key-..." -a josh-portfolio
fly secrets set MAILGUN_DOMAIN="mg.joshscompass.com" -a josh-portfolio
fly secrets set CONTACT_EMAIL="jpfajar90@gmail.com" -a josh-portfolio
```

#### How it works:

1. User fills out contact form on website
2. Frontend sends POST to `/api/contact`
3. Rust backend:
   - Saves submission to SQLite database
   - Spawns async task to send email via Mailgun API
4. Mailgun sends email to `jpfajar90@gmail.com`

#### Mailgun API Call (Rust/reqwest):

```rust
let url = format!("https://api.mailgun.net/v3/{}/messages", domain);

let form_params = [
    ("from", format!("Portfolio Contact <mailgun@{}>", domain)),
    ("to", recipient),
    ("subject", format!("{} - {}", subject, name)),
    ("text", format!("From: {} <{}>\n\n{}", name, email, message)),
];

client
    .post(&url)
    .basic_auth("api", Some(&api_key))
    .form(&form_params)
    .send()
    .await;
```
